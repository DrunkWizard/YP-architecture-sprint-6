Решение: 
- При создании заявки на ОСАГО паблишить событие в кафку 
- Сервис osago-aggregator читает кафку и сохраняет себе все заявки 
- Воркер отправляет все заявки в сервисы страховых раз в 5 секунд
- По всем заявкам поллятся методы страховых
- Все успешно обработанные заявки отправляются в кафку

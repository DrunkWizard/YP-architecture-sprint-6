Проблемы текущего решения:
- Синхронное запрашивание информации из 5ти страховых компаний и аггрегирование в момент запроса. 
    - Если сервис одной из страховых компаний по каким-то причинам упал - падает весь запрос 
    - Проблемы с согласованостью данных из-за того, что два сервиса в разное время получают и обновляют данные о тарифах 
    - Большая связность сервисов. Проблемы при изменении контрактов, падениях, задержках 
    - Разные системы страховых компаний могут по разному держать нагрузку и предъявлять свои SLA, а мы ходим ко всем одинаково
